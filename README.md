# Ejecutar en desarrollo

1. Clonar el repositorio
2. Ejecutar el comando 

```
yarn install
```

3. Tener instalado nest cli instalado
```
npm i -g @nestjs/cli
```

4.Levantar la bd
```
docker-compose up -d
```


## Stack usado

* Mongo db
* Nest
